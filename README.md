# Media Player
This is a web application created by Bilal Belli Â© 2022-2023. It allows you to add media elements (images or videos) and automatically displays these elements with timing and order, similar to a Media Player.

### Features
- Allow you to customize time and order of showing elements.
- Responcive design (for smartphones and desktops).
- Allow you to add, edit and delete media elements.
